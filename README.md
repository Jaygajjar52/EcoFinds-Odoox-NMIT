Tech Stack

Frontend: Flutter (Mobile App) / React (Web)

Backend: Firebase / Node.js (configurable)

Database: Firebase Firestore / MongoDB

Authentication: Firebase Auth (Email + Google)

Payments: Stripe / Razorpay / PayPal (demo-ready)
